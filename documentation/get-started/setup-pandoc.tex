\hypertarget{setup}{%
\section{Setup}\label{setup}}

From the Pandoc Latex Template download, copy \texttt{eisvogel.tex} to

\textasciitilde/.pandoc/templates/ (if this doesn't exist create it)

Rename \texttt{eisvogel.tex} to \texttt{eisvogel.latex}

\hypertarget{editing-the-document}{%
\section{Editing the Document}\label{editing-the-document}}

Markdown is easiest to use (you can use Sublime, VS Code etc) but you
can use LaTex editor with the .tex file.

Once you have finished editing the .md file or the .tex file, run the
respective command to compile everything (run them from the
documentation folder from git):

\hypertarget{create-.tex-.pdf-from-the-edited-the-.md-file}{%
\subsection{Create .tex \& .pdf from the edited the .md
file}\label{create-.tex-.pdf-from-the-edited-the-.md-file}}

\texttt{pandoc\ trekstar-documentation.md\ -o\ trekstar-documentation.tex\ -\/-template\ eisvogel}\\
\texttt{pandoc\ trekstar-documentation.md\ -o\ trekstar-documentation.pdf\ -\/-template\ eisvogel}

\hypertarget{create-.md-.pdf-from-the-edited-the-.tex-file}{%
\subsection{Create .md \& .pdf from the edited the .tex
file}\label{create-.md-.pdf-from-the-edited-the-.tex-file}}

\texttt{pandoc\ trekstar-documentation.tex\ -o\ trekstar-documentation.md\ -\/-template\ eisvogel}\\
\texttt{pandoc\ trekstar-documentation.tex\ -o\ trekstar-documentation.pdf\ -\/-template\ eisvogel}
